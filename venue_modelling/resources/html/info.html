User Guide

Log in

To create a new account type a username into the pop up box and click the "new user" button, to sign in again after you have created an account type in your user name and click on the "Log in" button

Uploading simulation parameters

Once logged in, in the bottom left of the screen, there is a table containing paraters used in the simulation of the room.

xLength, yLength, and zLength are the width, height, and length of the room respectively.

resolution is the amount of calculation points per meter so 10 would be 10 evenly spaced points every meter in all directions.

order is not exactly the max order of the reflections, but rather the order of the reflections in each cardinal direction x,y,z however, reflections in diagonal planes will be a higher order and are still calculated.

The next 6 parameters top, bottom etc. represent the absorption of each respective wall however, it is flat energy loss across the spectrum rather than frequency dependant absorption.

After you have set the parameters to the desired values, clicking the "Update" button will send those values to the database so the Algorithm can access them.

Uploading audio

In the bottom right of the screen there is a file input box, click on "Choose file" to open a file dialogue then click on "upload" to send the file to the server and begin the processing.

Before uploading you must ensure that the file you upload is a 16 bit PCM wav with a sample rate of 44,100 Hz.

Once the file has been uploaded, while you are logged in, the browser will ask the server every 5 seconds if processing has been completed. Once the simulation has been completed the browser will present you a button to download the output file.

Processing may take a while depending on the amount of points being simulated... (xLength * yLength * zLength) * resolution^3, however, you can close the web site and log back in at a later time and the browser will still be able to check for the simulatioin completing

Playing back a visualiser file

Above the parameter input area is another file input used for the playback of visualiser files, once the .vis.wav has been downloaded you can click on "choose file", select the file, then click on the "Load" button to load the file into the visualiser.

After loading the visualiser, clicking "play" will begin both audio and visualiser playback, the top-left box is the top down view of the room, the top-right box is the length ways view, and the bottom box is the length ways view.
